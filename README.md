## Spring Boot CQRS Example


### How to Run with docker:
> mvn clean install -P docker


> docker-compose -f docker-compose.yml up -d --build
### How to Run locally with development profile:
> mvn clean install -P development


> docker-compose -f docker-compose.yml up -d --build
